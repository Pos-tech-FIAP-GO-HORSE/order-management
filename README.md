# Order management

Order management service to serve customers more efficiently.

## Installation:

- From the root folder, run `docker-compose up` to install the required images.

- Create a `.env` file based off the `.env.example` on the root folder:

```
# API
APP_ENV=""

# Database
DB_STORAGE=""
DB_HOST=""
DB_PORT=""
DB_USER=""
DB_PASSWORD=""
DB_NAME=""

```
